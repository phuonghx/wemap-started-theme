# Install Sass


```sh
 npm install -g gulp
```

```sh
mkdir sass
```

```sh 
npm install gulp --save-dev
```

```sh 
npm install gulp-sass jshint gulp-jshint gulp-concat gulp-uglify gulp-rename gulp-minify-css gulp-livereload  gulp-clean-css --save-dev
```
# Run: 
```sh 
gulp
```
